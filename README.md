# Clon de Spotify

## Descripción del proyecto

Proyecto de maquetación **móvil y desktop** con menu hamburquesa desplegable en versopn de pantalla para moviles inspirado en la marca **Spotify**, con un menú interactivo **sidebar** en la versión móvil.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript

> La maquetación general de los elementos principales se hizo con **CSS Grid**, mientras que los elementos hijos se organizaron con **Flexbox**.

## Lo que aprendí

- Diferencia entre `justify-content` y `align-items`, y cómo usar correctamente `align-items` cuando `flex-direction` es `column`.
- TIP: Si no queda claro dónde están los márgenes de un componente, usar `style="border: 2px solid #color"` ayuda a visualizar sus límites y facilita la toma de decisiones sobre su diseño.

## Lo que mejoraría con más tiempo

- Reducir código repetido en CSS.
- Aclarar cuándo usar `<div>` y cuándo etiquetas semánticas como `<article>` o `<ul>`.
- Optimizar la estructura semántica del contenido.
- Sigo confundiendo cuando es conveniente usar px o %, me desbordan los margenes
